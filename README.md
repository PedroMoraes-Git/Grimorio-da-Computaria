## Grimório da Computaria 🔮

Minha documentação pessoal de programação. Código que funciona (talvez), alguns comentários sinceros e palavrões ocasionais.

**Começado em 14/03/2026. Estudando C#, .NET e outras bruxarias.**

### Aviso

Pode conter frustração genuína de dev, se isso te incomoda, esse repo não é pra você.

### O que tem aqui

- Processo de aprendizado de um dev;
- Vários exemplos de C#, .NET, Git e outros feitiços;
- Talvez código útil e soluções alternativas;
- Exemplos nem tão confiáveis se estiverem desatualizados.

### Pra quem é isso

Pra mim mesmo. Mas esse repositório é público por um motivo, então se quiser bisbilhotar ou ler o código, se sinta livre ^^

### Como usar

Sei lá. Entre nas pastas, copie o que quiser e talvez tente entender o código. Não quero créditos, simples assim.
